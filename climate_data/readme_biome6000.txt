DESCRIPTION:
This is the first release of the Biome6000 data. The data are organised 
according to publications. This first release only contains the most basic 
metadata (region, site name or code, latitude, longitude, and elevation) 
and the biome assignments for each of the sites, if available. 
A more detailed data report will be made available as soon as 
the regional data have come in.  Biome codes are as follows:

Code	Name
TUND	tundra
TAIG	taiga
CLDE	cold deciduous forest
CLMX	cold mixed forest
COCO	cool conifer forest
COMX	cool mixed forest
TEDE	temperate deciduous forest
TECO	temperate conifer forest
WAMF	broadleaved evergreen/warm mixed forest
TDFO	tropical dry forest/savanna
TSFO	tropical seasonal forest
TRFO	tropical rain forest
OC	open conifer woodland
XERO	xerophytic woods/scrub
SAVA	savanna
STEP	steppe
DESE	desert

Codes for Africa:
Code	Name
TXBS	Tropical xerophytic bush/savanna
WAGR	Warm grass/shrub
TDFO	Tropical dry forest/savanna
TSFO	Tropical seasonal forest
TRFO	Tropical rain forest
COGS	Cool grass/shrub
WAMF	Warm mixed forest
TEMS	Temperate microphyllous shrubland
TEXE	Temperate xerophytic woods/shrubs
